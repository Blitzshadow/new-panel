Client SaaS Dashboard
====================

Nowoczesny, customowy panel administracyjny dla klientów WordPress/WooCommerce.

Instalacja:
1. Skopiuj folder pluginu do wp-content/plugins
2. Aktywuj plugin w panelu WordPress
3. Upewnij się, że użytkownicy mają rolę "klient"

Funkcje:
- Panel klienta z sekcjami WooCommerce, wpisami, powiadomieniami, kontakt, zgłoszenia
- Nowoczesny, responsywny UI (TailwindCSS, Alpine.js)
- REST API/AJAX
- Bezpieczeństwo, personalizacja, rozbudowa
